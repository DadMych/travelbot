# Seed — старі подорожі

Редагуй `cities.seed.json` і запускай:

```bash
npm run seed
```

## Формат запису

```json
{
  "query": "Львів",
  "year": 2019,
  "notes": "опційно",
  "rating": 5
}
```

| Поле | Обов'язкове | Опис |
|------|-------------|------|
| `query` | так | Місто для пошуку. Краще з країною: `"Kraków, Poland"` |
| `year` | ні | Якщо пам'ятаєш тільки рік — постав його. Без року → «Дата невідома» |
| `notes` | ні | Замітка |
| `rating` | ні | 1–5 |

## Поради

- **Не пам'ятаєш дату** — просто не вказуй `year`, все ок
- **Плутаються міста** — додай країну в query: `"Paris, Texas"` vs `"Paris, France"`
- **Дублікати** — скрипт пропускає міста, які вже є на карті
- **Межі міст** — підтягнуться автоматично при відкритті сайту
- **Нові поїздки** — далі через Telegram-бота (там дата = сьогодні)

```json
  { "query": "Барcelona, Spain", "year": 2022, "rating": 5 }
]
```
